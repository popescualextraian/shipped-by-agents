# Quick Paths

Most developers don't have time to read through full training chapters. Quick Paths are the fast lane — short, actionable recipes that get you productive with AI agents in minutes, not hours.

Each quick path gives you **what to do, how to set it up, and where to go next**. No deep theory, no long reads. Every section links to official docs and to our [technical chapters](../technical/) when you want to dig deeper.

## How each quick path is structured

Every activity within a quick path follows the same format:

- **What** — one or two sentences explaining the activity
- **Setup** — what you need to install or configure
- **How** — step-by-step recipe (3-5 steps)
- **Links** — official docs + the matching chapter from our [technical section](../technical/)

At the end of each quick path you'll find a **Tools & Alternatives** table — a single view of what to install and what skill bundles (like Superpowers) can enhance each activity.

## Who this is for

Developers who want to start using AI agents **now**. You don't need to understand the theory first — start here, get results, then come back to the [technical chapters](../technical/) when you want to understand the why behind the how.
