## Who this is for

Developers and teams who want to understand, adopt, and get productive with AI coding agents — tools like Claude Code, Cursor Agent, GitHub Copilot, and Devin.

No prior AI experience required. Just bring your editor and curiosity.

## How to use this

- **Self-study** — read the chapters in order, try the examples
- **Workshop** — present to your team, one or two chapters per session
- **Reference** — jump to any chapter; each one is self-contained
