# The Shared Repository

The nervous system of the Shipped by Agents framework. Ideas flow up from teams, solutions flow down from the Forge.

See [The Process](../process/00_shipped-by-agents-process/00_shipped-by-agents-process.md#the-shared-repository) for full context.

---

## Structure

| Folder | What goes in | Who contributes |
|--------|-------------|-----------------|
| `ideas/` | Automation opportunities discovered by teams — raw ideas, not polished proposals | Crews, Pathfinders |
| `patterns/` | Recurring patterns identified across teams (e.g., "three teams need the same Jira workflow") | Pathfinders, Bridge |
| `solutions/` | Reusable prompts, skills, agent configurations, and production-ready tools built by the Forge | Pathfinders, Crews, Forge |
| `lessons-learned/` | What failed and why — post-mortems, anti-patterns, dead ends | All teams |
